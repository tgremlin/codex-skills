# Product Spec: CRUD Basic

## Goal

Ship a lightweight full-stack todo tracker with create/list/update/delete.

## Scope

- Users can create, list, update, and delete todos.
- UI reflects the latest backend state after each mutation.
- Data persists across service restarts.

## Acceptance Criteria

1. User can create a todo.
2. User can update a todo completion state.
3. User can delete a todo.
4. Data persists after restart.
