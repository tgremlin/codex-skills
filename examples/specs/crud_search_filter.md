# Product Spec: CRUD With Search And Filter

## Goal

Add searchable and filterable todo listing on top of CRUD.

## Scope

- Users can create, update, and delete todos.
- Users can search todos by title.
- Users can filter by completion state.
- Backend returns filtered/search results from persisted data.

## Acceptance Criteria

1. User can create multiple todos and view all items.
2. User can search by title and get matching todos only.
3. User can filter by completed vs active state.
4. User can combine search and filter in one request.
